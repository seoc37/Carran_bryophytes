This website contains material from the Carran bryophytes project, 2020, part of the M.Sc. in Biodiversity and Conservation at Trinity College, University of Dublin.

Everything is hosted in a repository on GitHub.

The project thesis will be available here (xx MB PDF). [awaiting upload, Jan 2023]

High resolution versions of the maps from the report are here. They are linked from the PDF document for easy access in context.


Project data are available from GitHub at https://seoc37.github.io/Carran_bryophytes. Follow the README files to locate the data files, spreadsheets, analysis outputs and QGIS maps.

The GitHub repository is https://github.com/seoc37/Carran_bryophytes for access using git.
